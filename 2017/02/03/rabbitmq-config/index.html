管理

我们可以直接通过配置文件的访问进行管理，也可以通过Web的访问进行管理。以下是开启Web UI进行管理的方法。

  1. 执行rabbitmq-plugins enable rabbitmq_management命令，开启Web UI管理插件
  2. 访问 http://localhost:15672 进行RabbitMQ管理的登陆页面、默认的用户名及密码为均为guest、登陆后进行到RabbitMQ的概述页面
